# ADR-002 — Abstraksi Service Layer Wajib

## Status

Diterima.

## Context

- Memanggil Appwrite SDK langsung dari halaman menyebar logika akses data ke seluruh UI.
- Hal ini mempersulit pengujian, penggantian implementasi, dan menjaga konsistensi query/permission.

## Decision

Seluruh akses Appwrite **wajib** melalui service layer:

```text
Page → Service → Appwrite SDK
```

Halaman tidak boleh memanggil Appwrite SDK secara langsung.

## Consequences

- (+) UI terisolasi dari detail SDK; mudah diuji dan diganti.
- (+) Logika query/permission terpusat dan konsisten.
- (−) Ada lapisan tambahan yang harus dibuat untuk setiap domain (authService, campaignService, dll.).
- Ditegakkan sebagai aturan kerja: [50_Project_Rules.md](../00_Project/50_Project_Rules.md).
