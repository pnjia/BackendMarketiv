# 99_Templates — Index

Template kosong yang dapat dipakai ulang untuk menjaga konsistensi struktur Knowledge Base. Salin file yang relevan, ganti placeholder `<...>`, hapus baris panduan.

## Daftar Template

- [Module_Template.md](Module_Template.md) — Saat membuat modul baru di `02_Modules/`. Mendeskripsikan set dokumen standar (00–100).
- [Workflow_Template.md](Workflow_Template.md) — Saat mendokumentasikan alur lintas-modul di `03_Workflows/`.
- [ADR_Template.md](ADR_Template.md) — Saat mencatat keputusan arsitektur di `04_Decisions/`.
- [API_Template.md](API_Template.md) — Saat mendokumentasikan satu method service (dipakai di `60_API.md` modul).
- [Database_Template.md](Database_Template.md) — Saat mendokumentasikan satu collection (dipakai di `50_Database.md` modul).
- [Business_Rules_Template.md](Business_Rules_Template.md) — Saat menulis aturan bisnis modul (`30_Business_Rules.md`).
- [Screen_Template.md](Screen_Template.md) — Saat mendokumentasikan satu screen/route (dipakai di `80_Frontend.md` modul).
