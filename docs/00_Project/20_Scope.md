# Ruang Lingkup MVP

## Termasuk dalam MVP

- Authentication (login, register UMKM/Creator, Google OAuth untuk creator, forgot password)
- UMKM Profile
- Creator Profile
- Rate Card
- Campaign (Pay-Per-View)
- AI Brief Generator
- Offer System
- Chat dasar untuk negosiasi Offer (termasuk attachment terbatas untuk referensi negosiasi)
- Order System
- Escrow
- Submission
- AI Fraud Detection
- Approval Workflow
- Wallet
- Withdraw
- Notification (In-App + Email)

## Dikecualikan dari MVP

- Email Verification
- Chat Real-time tingkat lanjut, termasuk typing indicator, read receipt, multi-file upload, attachment kompleks, voice note, moderation otomatis, dan fitur chat sosial lainnya.
- Affiliate System
- Referral
- Creator Agency
- Team Management
- Subscription
- AI Matching
- Marketplace Bidding
- Public Campaign Board
- Analytics Advanced
- Mobile App
