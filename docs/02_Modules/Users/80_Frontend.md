# Users — Frontend

## Halaman

### Profile UMKM

- Tampilan profil bisnis (nama, kategori, deskripsi, kota, kontak, logo).
- Tombol Edit Profile.

### Profile Creator

- Tampilan profil (display name, bio, avatar, rating, followers).
- Daftar akun sosial (platform, username, followers).
- Daftar portofolio.
- Rate card (dari modul RateCards).
- Tombol Chat.

### Edit Profile

- Form dinamis sesuai role.
- Upload logo/avatar.
- Tambah/hapus akun sosial (Creator).
- Tambah/hapus portofolio (Creator).

### Browse Creator

- Grid/list card Creator.
- Filter: platform, kota.
- Search bar.

## Komponen

- `ProfileCard` — ringkasan profil.
- `SocialAccountList` — daftar akun sosial.
- `PortfolioGrid` — grid portofolio.
- `CreatorCard` — card untuk halaman discovery.
- `SearchFilter` — filter pencarian Creator.
