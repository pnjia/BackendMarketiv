# Campaigns — Concepts

## Istilah

- **Campaign** — proyek promosi UMKM dengan sistem Pay-Per-View (PPV).
- **Pay-Per-View (PPV)** — model pembayaran di mana creator dibayar berdasarkan jumlah views konten.
- **Claim** — aksi creator mengambil campaign untuk dikerjakan.
- **Submission** — konten yang diunggah creator sebagai hasil kerja campaign.
- **Fraud Check** — analisis otomatis untuk memvalidasi keaslian submission.
- **Reward** — kompensasi untuk creator = `(views / 1000) × rewardPer1000Views`.
- **CPM** — Cost Per Mille (per seribu views).

## Status Campaign

`draft → active → paused → completed`

## Status Claim

`claimed → submitted → approved → rejected`

## Status Submission

`pending → approved | rejected`
