# Offers — Frontend

## Komponen

### OfferForm

- Ditampilkan di dalam Chat (modal/bottom sheet).
- Hanya untuk UMKM.
- Fields: title, description, price, deadline, revisionLimit.

### OfferCard

- Ditampilkan sebagai pesan tipe `offer` di chat.
- Menampilkan semua field offer.
- Tombol Accept / Reject (untuk Creator).
- Status badge (pending/accepted/rejected).

### OfferStatusBadge

- `pending` — abu-abu / biru.
- `accepted` — hijau.
- `rejected` — merah.
