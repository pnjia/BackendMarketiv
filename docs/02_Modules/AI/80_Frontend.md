# AI — Frontend

## Komponen

### AI Brief Generator

- Tombol "Generate Brief" di halaman Create/Edit Campaign.
- Loading state selama AI memproses.
- Menampilkan hasil brief dalam form yang dapat diedit.
- Tombol "Simpan Brief" untuk menyimpan ke `campaign_briefs`.

### Fraud Detection (tidak ada UI khusus)

- Berjalan otomatis di backend.
- Hasil fraud ditampilkan sebagai badge/label pada submission di dashboard UMKM:
  - `safe` — hijau
  - `review` — kuning
  - `rejected` — merah
