
# 5. Sprint Breakdown & Development Roadmap

Karena Marketiv masih tahap MVP dan tim hanya 4 orang, saya sarankan menggunakan **6 Sprint (12 minggu)** dengan target menghasilkan MVP yang siap digunakan oleh UMKM dan kreator di Sukabumi.

---

# Sprint 1 — Foundation & Authentication

Durasi: 2 Minggu

### Target

Membangun pondasi aplikasi.

### Scope

#### Backend

* Setup Appwrite
* Authentication
* Role System

  * UMKM
  * Creator
  * Admin
* Wallet otomatis saat registrasi
* Upload avatar
* Upload media dasar

#### Frontend

* Landing Page
* Login
* Register
* Onboarding

### Deliverable

* User bisa daftar
* User bisa login
* Dashboard kosong sudah bisa diakses

---

# Sprint 2 — Creator Profile & Rate Card

Durasi: 2 Minggu

### Target

Creator dapat membuat profil profesional.

### Scope

#### Creator

* Edit profile
* Social media account
* Category/Niche
* Portfolio
* Rate Card
* Verification badge request

#### UMKM

* Creator discovery
* Search creator
* Filter creator

### Deliverable

* Marketplace Creator sudah berjalan
* UMKM dapat melihat profil creator

---

# Sprint 3 — Campaign Module (Core Viral Feature)

Durasi: 2 Minggu

### Target

Fitur utama Campaign Pay Per View.

### Scope

#### UMKM

* Create Campaign
* Upload Raw Video
* AI Brief Assistant
* Budget Setup

#### Creator

* Browse Campaign
* Claim Campaign
* Submit Result

#### System

* Campaign Status
* Submission Status

### Deliverable

* Campaign dapat dibuat
* Creator dapat claim
* Creator dapat submit

---

# Sprint 4 — Order, Offer & Escrow

Durasi: 2 Minggu

### Target

Fitur Rate Card benar-benar usable.

### Scope

#### Rate Card Flow

* Chat Room
* Custom Offer
* Offer Accept
* Order Creation
* Escrow Creation
* Revision System
* Delivery System

#### Wallet

* Deposit
* Withdraw Request
* Transaction History

### Deliverable

* UMKM dapat booking creator
* Escrow berjalan

---

# Sprint 5 — AI Fraud & Analytics

Durasi: 2 Minggu

### Target

USP utama Marketiv.

### Scope

#### Fraud Detection

* URL submission verification
* View tracking
* Suspicious pattern detection
* Fraud score

#### Analytics

UMKM:

* Views
* Spend
* CPM
* Creator Performance

Creator:

* Earnings
* Success Rate
* Campaign Performance

### Deliverable

* View tracking berjalan
* Dashboard analytics tersedia

---

# Sprint 6 — Admin Panel & Production Ready

Durasi: 2 Minggu

### Target

MVP siap digunakan publik.

### Scope

#### Admin

* User Management
* Campaign Management
* Order Management
* Withdrawal Approval
* Dispute Resolution
* Fraud Monitoring

#### Production

* Notification
* Email
* Error Logging
* Audit Log
* Security Review
* Performance Optimization

### Deliverable

* MVP siap soft launching
